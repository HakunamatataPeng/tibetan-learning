# 📿 藏语学习助手

一个纯前端的藏语学习工具，支持手动录入单词、闪卡背单词、以及基于录音的口语练习。所有数据保存在浏览器本地，无需服务器。

## 功能特性

### 📝 单词管理
- 手动录入藏语单词、威利转写、发音提示、中文翻译
- 支持分类标签（日常用语、数字、颜色、家人、食物、动词、形容词等）
- 可添加例句和备注
- 搜索、筛选、编辑、删除单词

### 🎴 背单词（闪卡模式）
- 三种学习模式：藏语→中文、中文→藏语、随机
- 点击翻转卡片，查看答案
- 支持按分类筛选学习内容
- 记录掌握状态和学习进度
- 学习连续天数统计

### 🎤 口语练习
- 上传老师的录音文件（MP3、WAV、OGG、M4A）
- 支持拖拽上传和批量上传
- 三种练习模式：
  - **听读模式**：循环播放、慢速播放
  - **影子跟读**：自动间隔暂停辅助
  - **录音对比**：录制自己的发音并与老师对比

## 使用方法

1. 用浏览器打开 `index.html` 文件（推荐使用 Chrome 或 Edge）
2. 在「单词管理」页面录入藏语单词
3. 在「背单词」页面选择模式和分类，开始学习
4. 在「口语练习」页面上传老师的录音，进行跟读练习

## 数据说明

- **单词数据**：保存在浏览器 localStorage 中
- **录音文件**：保存在浏览器 IndexedDB 中
- 清除浏览器数据会导致学习记录丢失，建议定期导出备份

## 藏语字体

应用使用 Google Fonts 的 **Noto Sans Tibetan** 字体显示藏文，需要联网加载。如果无法联网，浏览器会回退到系统藏文字体。

## 技术栈

- HTML5 + CSS3 + 原生 JavaScript
- IndexedDB（录音文件存储）
- localStorage（单词数据存储）
- Web Audio API（录音功能）

## 本地开发

无需构建步骤，直接用浏览器打开 `index.html` 即可。

```bash
# 如果用本地服务器（可选）
python -m http.server 8080
# 然后访问 http://localhost:8080
```

## 注意事项

- 录音功能需要浏览器授权麦克风权限
- 由于浏览器的安全限制，音频文件只能通过用户主动上传，不能直接访问本地文件系统
- 建议定期将录音文件在本地做好备份
